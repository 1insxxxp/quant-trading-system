# 实时价格双层标签无缝拼接设计

**日期：** 2026-03-15

## 背景

当前右侧实时价格已经由图表原生价格轴标签渲染，倒计时也已经放在价格正下方并复用了同色逻辑。剩余问题是两块在视觉上还没有拼成一个整体：

- 倒计时块宽度没有严格铺满，与价格块不完全同宽
- 上下两块之间还没有做到无缝拼接

## 目标

- 保持上方价格继续由原生价格轴标签渲染
- 下方倒计时块与上方价格块同宽
- 左右边缘完全对齐
- 上下两块之间无缝拼接

## 方案

### 结构

不改结构，仍然是：

- 原生价格轴标签
- 自绘倒计时块

### 样式

通过当前已经有的 `--chart-price-scale-width`，让倒计时块：

- 宽度占满整个价格轴标签容器
- 使用 `box-sizing: border-box`
- 去掉额外横向收缩
- 通过固定竖向偏移，把块顶边贴到原生价格标签底边附近

### 约束

不恢复自绘价格，也不把价格标签改回 DOM。只修倒计时块的几何关系。

## 涉及文件

- `frontend/src/index.css`
- `frontend/src/index.css.test.js`

## 验证标准

- 倒计时块宽度铺满价格轴容器
- 倒计时与价格块左右对齐
- 两块之间无明显缝隙
- `npm test -- --run src/index.css.test.js`
- `npm run build`
